# Resource Efficiency and Process Optimization - Sankey Diagram

**Author Email:** 22f3003001@ds.study.iitm.ac.in

## Project Overview

Professional Sankey Diagram created using RAWGraphs to visualize resource flow and process optimization in a manufacturing environment.

## Business Context

This visualization demonstrates:
- **Resource Allocation**: How resources flow through different processes
- **Efficiency Analysis**: Identification of bottlenecks and waste
- **Process Optimization**: Visual representation of production pathways

## Sankey Diagram Details

The diagram illustrates the flow of resources through:

1. **Input Sources** → Raw materials and resources
2. **Processing Stages** → Manufacturing and transformation processes
3. **Output Destinations** → Final products and waste streams

### Flow Categories

- **Primary Flow**: Main production pathway
- **Secondary Flow**: Recycling and reprocessing
- **Waste Flow**: Inefficiencies and waste products

## Visualization Specifications

- **Tool:** RAWGraphs (Web-based)
- **Chart Type:** Sankey Diagram
- **Dimensions:** 400x400 to 512x512 pixels
- **Format:** PNG
- **Color Scheme:** Professional gradient showing flow intensity

## Data Structure

The Sankey diagram is based on source-target-value relationships:

```
Source → Target → Value
Raw Materials → Processing → 1000
Processing → Assembly → 800
Processing → Waste → 200
Assembly → Product A → 500
Assembly → Product B → 300
```

## Key Insights

- **Efficiency Rate**: 80% of raw materials converted to products
- **Waste Reduction Opportunity**: 20% of resources lost in processing
- **Optimization Potential**: Recycling pathways identified

## How to Create

1. Visit [RAWGraphs.io](https://app.rawgraphs.io/)
2. Upload CSV data with source, target, and value columns
3. Select "Alluvial Diagram" or "Sankey Diagram" chart type
4. Map data columns to visual elements
5. Customize colors and styling
6. Export as PNG (400-512px dimensions)
